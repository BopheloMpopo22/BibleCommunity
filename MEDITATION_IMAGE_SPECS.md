## üìê **Recommended Image Specs**

### **1. Category Grid Images (Title Boxes)**

**Current Usage:**

- Displayed in 2-column grid
- Card size: ~180px height, ~50% screen width
- Used for category selection screen

**Recommended Specs:**

- **Resolution:** 800x800px (square) or 1200x800px (landscape)
- **Aspect Ratio:** 1:1 (square) or 3:2 (landscape)
- **File Format:** JPG or PNG
- **File Size:** Under 500KB (optimized)
- **Quality:** High quality, but compressed for web/mobile

**Why These Specs:**

- Square works best for grid cards
- 800px width is enough for mobile displays
- Small file size = fast loading
- Landscape also works but square is preferred

**Pexels URL Format:**

```
https://images.pexels.com/photos/[ID]/pexels-photo-[ID].jpeg?auto=compress&cs=tinysrgb&w=800
```

---

### **2. Fullscreen Meditation Images**

**Current Usage:**

- Full screen background during meditation
- Covers entire screen
- Has dark overlay (30% opacity)

**Recommended Specs:**

- **Resolution:** 1920x1080px (Full HD) or 2560x1440px (2K)
- **Aspect Ratio:** 16:9 (landscape)
- **File Format:** JPG (preferred) or PNG
- **File Size:** Under 1MB (optimized)
- **Quality:** High quality, optimized for mobile

**Why These Specs:**

- 16:9 matches most phone screens
- 1920x1080 is standard Full HD
- Large enough for all screen sizes
- Compressed to keep app size reasonable

**Pexels URL Format:**

```
https://images.pexels.com/photos/[ID]/pexels-photo-[ID].jpeg?auto=compress&cs=tinysrgb&w=1200
```

**For Local Files:**

- Save as: `meditation-love-bg.jpg`
- Resolution: 1920x1080px
- File size: ~500KB-1MB

## üì± **Image Optimization Tips**

### **1. Compression:**

- Use tools like TinyPNG or ImageOptim
- Target: 70-80% quality
- File size: Under 1MB for fullscreen, under 500KB for grid

### **2. Format:**

- **JPG:** Best for photos (smaller file size)
- **PNG:** Best for graphics (larger file size)
- **WebP:** Best compression (if supported)

### **3. Dimensions:**

- **Category Grid:** 800x800px (square) or 1200x800px
- **Fullscreen:** 1920x1080px (16:9) minimum

### **4. Loading:**

- Use CDN or optimized URLs
- Consider lazy loading for grid
- Preload fullscreen images
