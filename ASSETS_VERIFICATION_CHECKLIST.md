**File Requirements:**
- [ ] **Size**: Exactly 1024 x 1024 pixels
- [ ] **Format**: PNG file (not JPG)
- [ ] **No Transparency**: Icon must have a solid background (iOS requirement)
- [ ] **File Size**: Under 5MB (usually icons are much smaller)

**Design Quality:**
- [ ] **Readable at Small Size**: Test by zooming out to 48x48 pixels - can you still see what it is?

- [ ] **High Quality**: Sharp, not blurry or pixelated
- [ ] **Brand Colors**: Uses your app's color scheme (#1a365d)
- [ ] **Professional**: Looks polished and professional

**Testing:**
- [ ] **Preview on Phone**: Replace icon in app and see how it looks on device
- [ ] **Test in Different Sizes**: Check how it appears at various sizes

### ğŸ“¸ Screenshots Verification (6 Screenshots)

**File Requirements:**
- [ ] **Correct Dimensions**: 
  - **iOS**: 1290 x 2796 pixels (iPhone 6.7") OR 1242 x 2688 pixels (iPhone 6.5")
  - **Android**: 1080 x 1920 pixels minimum (or higher)
- [ ] **Format**: PNG or JPG (PNG preferred for quality)
- [ ] **File Size**: Each screenshot under 10MB
- [ ] **Orientation**: All screenshots should be portrait (vertical)

**Content Quality:**
- [ ] **No Personal Information**: 
  - No real email addresses visible
  - No real names visible (unless you want to show them)
  - No phone numbers
  - Blur or remove any sensitive data
- [ ] **Real Content**: Shows actual app features, not empty screens
- [ ] **Clean UI**: No debug messages, error states, or placeholder text
- [ ] **Key Features Shown**: Each screenshot highlights a different feature
- [ ] **Professional Look**: Polished, consistent design across all screenshots

**Screenshot Content Checklist:**
- [ ] Screenshot 1: **Prayer/Daily Content** - Shows beautiful prayer screen
- [ ] Screenshot 2: **Community** - Shows active community with posts
- [ ] Screenshot 3: **Bible Study** - Shows scripture reading
- [ ] Screenshot 4: **Daily Scripture/Word** - Shows daily content
- [ ] Screenshot 5: **Settings/Profile** - Shows user account
- [ ] Screenshot 6: **Additional Feature** - Meditation, Bookstore, or Testimonies

**Design Consistency:**
- [ ] **Same Style**: All screenshots have consistent editing style
- [ ] **Text Overlays** (if used): Same font, size, and placement
- [ ] **Colors**: Consistent color scheme across all screenshots

## âš ï¸ Common Mistakes to Avoid

1. **Icon too complex** - Won't be readable at small sizes
2. **Screenshots with personal info** - Privacy issue
3. **Wrong dimensions** - App stores will reject
4. **Low quality images** - Looks unprofessional
5. **Empty screens** - Doesn't show app value
6. **Inconsistent editing** - Looks unprofessional
7. **Text overlays too small** - Can't read on small screens
